소개

운영체제·임베디드 과제에서 시작해, 지금은 SSAFY 모바일 트랙에서 자바·스프링·코틀린 기반 앱까지 확장해 나가고 있습니다.

학부에서 C 기반 임베디드 과목과 운영체제 과제를 수행하며 메모리, 프로세스, 드라이버를 직접 구현해 봤습니다. S32K144 보드로 LED 리듬 게임을 만들고, custom malloc을 작성해 보는 과정에서 하드웨어와 OS가 어떻게 움직이는지 몸으로 익혔습니다.

현재는 SSAFY 모바일 트랙을 수강하며 Java, SQL, Spring, Kotlin을 중심으로 공부하고 있습니다. 팀 프로젝트로 꽃 주문 제작 앱을 개발하며 백엔드 API 설계와 안드로이드 화면 구현을 동시에 경험하고 있습니다.

현재 목표

하드웨어와 OS를 이해하는 기반 위에 모바일·백엔드 역량을 더해, 임베디드와 서비스를 모두 이해하는 개발자로 성장하는 것이 목표입니다.
